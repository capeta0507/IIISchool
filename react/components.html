react Components 簡易語法練習

React.Component是一個抽象基類。這意味著直接引用React.Component是毫無意義的。你可以實現一個它的子類，並且至少定義一個render()方法即可使用。

  1. 先將react的JS連結引用至html網頁
  2. 在body設div並做id命名
  3. 在script標籤裡寫下react程式語法
  4. 先用class開頭並命名後段加上extends React.Component
  5. 括號並加上render()，括號並加上return 並寫上jsx語法(直接寫上html語法)
  6. 接著下方輸入ReactDOM.render()
  7. 括號內將class的命名以html標籤形式輸入逗號加上預設變數
  8. 若有多項標謙需在外層加上div包起來

React 參考網址

props是react的屬性。props 通常用在靜態資料上，讓父元件傳遞資料給子元件，而元件內部可以透過this.props 取得該值。

  1. 在props後加上一個命名
  2. 接著在reactDOM內的標籤內加上命名並寫上內容
