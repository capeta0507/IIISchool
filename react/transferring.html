傳遞props 語法練習

React 裡有一個非常常用的模式就是對組件做一層抽象。組件對外公開一個簡單的屬性（Props）來實現功能，但內部細節可能有非常複雜的實現。

  1. 先將react的JS連結引用至html網頁
  2. 在body設div並做id命名
  3. 在script標籤加上type="text/babel"，內部寫下react程式語法
  4. 準備3個class命名
  5. 在第一層預設變數並加上值
  6. 每往上一層用p標籤加上{this.props.變數}可將值帶入
