Git+Github Push+Pull

重要 Git 指令

  1. 在Local端建立第一個版本
  2. git push -u origin master:推送到github主機
  3. Settings/Collaborators/Add collaborator:邀請對方加入合作
  4. git pull:拉回新的的內容
  5. git push:推上新的內容(git add / commit 指令之後)

段落重點:邀請合作對象
