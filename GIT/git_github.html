Git+Github Push+Pull

重要 Git+Github 指令

  1. 在Local端建立第一個版本(add + commit)
  2. git push -u origin master:推送到github主機
  3. Settings/Collaborators/Add collaborator:邀請對方加入合作
  4. git pull:拉回新的的內容(git push = git fetch + git merge)
  5. git push:推上新的內容(git add / commit 指令之後)= git push origin master
  6. git push -f 就是 --force(強制)，會強迫硬推上去，覆蓋 GitHub上面的版本
  7. git clone git@github.com:kaochenlong/dummy-git.git 複製一份回Local
  8. git clone git@github.com:kaochenlong/dummy-git.git hello_kitty 克隆下來之後指定 hello_kitty目錄

段落重點:邀請合作對象
