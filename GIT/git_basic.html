Git 版本控管

重要 Git 指令

  1. Git init : 該資料夾建立一個 Git Repository
  2. Git add index.html:把index.html加入暫存區
  3. Git add --all (Git add .):將該管理資料夾全部加入
  4. Git status:查看 Git 最新狀態
  5. Git commit -m "5/30更新":把暫存區的紀錄提交到歷史紀錄
  6. Gitk:GUI工具
  7. Git config --global core.editor notepad:Git commit 指令不帶 -m ，會指定editor
  8. Git diff index.html:比對index.htm前後修改狀況
  9. Git log --pretty=oneline 檢視git歷史紀錄LOG，單行顯示
  10. Git reset --hard HEAD^:回復前一版本，HEAD^^就回到前前一個版本=HEAD~2
  11. Git reset --hard 3628164:回復 commit id是“3628164...”的版本
  12. Git reflog:紀錄你的每一次命令
  13. Git show HEAD:顯示最新 Commit 的詳細記錄
  14. git reset index.html:Git add index.html 尚未Commit 之前可以取消 Git add 狀態
  15. Git checkout 3628164 -- index.html:針對 index.html這個文件，回復到commit id是“3628164...”的版本
  16. .gitignore:本身是一個文字檔，輸入不想被版本控制的檔案名稱
