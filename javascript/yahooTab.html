練習 Tab 顯示技巧

  1. 以 button tab 設定顯示分類標籤
  2. 每一分類 Tab 標籤獨立顯示
  3. Tab內容都已經寫在 HTML 文件裡，設定獨立 div + id name 並且不顯示
  4. 當 Tab button Click 時啟動 ShowPage function , 再給予Class = Active
  5. div + id name 增加 Acive Class name 之後就予以顯示
  6. button tab 也加入 Active class name 予以變更顯示效果
  7. 相關事件與顯示都用 javascript 控制
