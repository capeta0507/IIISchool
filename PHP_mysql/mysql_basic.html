SQL語法:結構化查詢語言

  1. C新增:INSERT INTO tableName (fieldName1, fieldName2,...) VALUES('Gates',35,...)
  2. R讀取:SELECT fieldName1,fieldName2,... FROM tableName;
  3. U修改:UPDATE tableName SET fieldName1 = 'Bill' WHERE id=1
  4. D刪除:DELETE FROM tableName WHERE id=1
  5. 條件:WHERE
  6. 唯一:DISTINCT
  7. 排序:ORDER BY
